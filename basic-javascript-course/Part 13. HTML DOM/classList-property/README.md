# ClassList Property

```
- Là thuộc tính của element node
- Khi element.classList trả lại 1 đối tượng webAPI ( DOMTokenList )

- Một số phương thức:
    + add: Thêm class vào element, thêm nhiều class cách nhau bởi dấu ,
    + contains: Kiểm tra xem 1 class có nằm trong element này hay không
    + remove: Xóa bỏ class khỏi element
    + toggle: Nếu có class thì nó xóa và ngược lại - Ứng dụng làm menu 
```
