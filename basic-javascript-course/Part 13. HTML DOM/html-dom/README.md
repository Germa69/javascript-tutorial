# HTML - DOM

```
- DOM: Viết tắt của từ Document Object Model = mô hình tài liệu thể hiện dưới dạng đối tượng
- Khi trang web được tải, trình duyệt tạo ra DOM cho trang web 

- Trình duyệt nhận 1 chuỗi HTML, từ chuỗi HTML dựa trên quy chuẩn W3C được cài đặt sẵn dưới thằng trình duyệt sau đó sẽ dịch chuỗi đó thành DOM
```

```
- DOM có 3 thành phần: 
1. Element
Những thẻ tag viết trong file HTML đều là element

2. Attribute
Là những thuộc tính nằm trong element

3. Text
```

# Javascript

```
- HTML - DOM không phải là của JavaScript
- Là công cụ để truy xuất vào HTML - DOM
- Chạy ở Browser (Chrome, morila, ...) | Server (NodeJS)

- Browser: HTML -> DOM -> DOM API (Application programing interface)

- DOM nằm trong WEB API => Chỉ có trong nền tảng hỗ trợ mở được trang web  
```
