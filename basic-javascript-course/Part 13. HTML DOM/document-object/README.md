# Document Object 

```
- Tất cả các elememt, attribute, text đều là thành phần nhỏ hơn Document vì vậy nó là đại diện cho cả website.
- Khi trình duyệt đọc hết tất cả mã HTML nó sẽ tạo ra 1 DOM
- Khi tạo ra DOM thì nó sẽ lưu hết vào biến Document
- Để truy cập được element, attribute, text sẽ phải luôn luôn đi qua đối tượng Document
- Được tích hợp sẵn trong WEB API
```
